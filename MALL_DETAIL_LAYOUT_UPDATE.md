# 商品详情布局调整

## 修改内容
将商品详情对话框中的图片展示区域从左侧移到右侧。

## 布局变化

### 之前的布局：
```
┌─────────────────────────────────────┐
│  图片展示  │  商品信息（价格、规格）│
└─────────────────────────────────────┘
```

### 现在的布局：
```
┌─────────────────────────────────────┐
│  商品信息（价格、规格）│  图片展示  │
└─────────────────────────────────────┘
```

## 技术实现
使用 CSS Flexbox 的 `order` 属性调整元素顺序：

```css
.detail-info {
  order: 1;  /* 显示在左侧 */
}

.detail-gallery {
  order: 2;  /* 显示在右侧 */
}
```

## 优势
- 用户先看到商品信息（标题、价格、规格），更符合阅读习惯
- 图片作为辅助信息放在右侧
- 不改变 HTML 结构，只通过 CSS 调整顺序

## 文件修改
- `frontend/src/views/Mall.vue` - 调整 detail-info 和 detail-gallery 的显示顺序
