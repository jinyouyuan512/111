# ⚠️ 前端端口变更通知

## 📊 重要信息

### 端口变更
- ❌ 原计划端口：3001
- ✅ 实际运行端口：**3002**

### 原因
端口 3001 已被其他进程占用，Vite 自动切换到了 3002。

## 🚀 正确的访问地址

### 前端服务
```
http://localhost:3002
```

### 完整服务列表
```
✅ Social Service   → http://localhost:8083 (文件上传和存储)
✅ Creative Service → http://localhost:8087 (作品数据管理)
✅ Frontend         → http://localhost:3002 (用户界面) ⚠️ 注意端口
```

## 🔧 如果需要使用端口 3001

### 方法 1：停止占用 3001 的进程

#### 查找占用进程
```powershell
Get-NetTCPConnection -LocalPort 3001 | Select-Object LocalPort, OwningProcess
```

#### 停止进程
```powershell
Stop-Process -Id <进程ID> -Force
```

#### 重启前端
停止当前前端进程（Ctrl+C），然后重新运行：
```bash
cd frontend
npm run dev
```

### 方法 2：直接使用 3002
不需要修改任何配置，直接使用 3002 端口即可。

## 📝 注意事项

1. **Vite 代理配置仍然有效**
   - `/uploads` 仍然代理到 `http://localhost:8083`
   - 所有 API 请求仍然正常工作

3. **书签和收藏**
