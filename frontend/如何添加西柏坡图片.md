## 📋 操作步骤

### 步骤 1：保存图片文件

1. **右键点击聊天中的图片**
2. **选择"图片另存为..."**
3. **保存位置：** 选择项目目录
   ```
   D:\ruler\frontend\public\images\hero\
   ```
4. **文件名：** 输入 `xibaipo.jpg`
5. **点击"保存"**

### 步骤 2：验证文件

确认文件已保存到正确位置：
```
frontend/public/images/hero/xibaipo.jpg
```

### 步骤 3：启用本地图片

打开文件：`frontend/src/views/Home.vue`

找到大约第 638 行的代码：
```javascript
const USE_LOCAL_IMAGES = false
```

修改为：
```javascript
const USE_LOCAL_IMAGES = true
```

### 步骤 4：保存并查看效果

1. 保存 `Home.vue` 文件
2. 浏览器会自动刷新
3. 访问 http://localhost:3000/
4. 查看首页轮播，西柏坡图片应该显示了！
