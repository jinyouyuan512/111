## 数据存储

### localStorage 键值
- `sceneProgress` - 场景进度（JSON对象）
- `scene_{sceneId}_interactions` - 交互点完成状态（数组）

### 数据结构

**场景进度**
```json
{
  "1": 100,
  "2": 50,
  "3": 0,
  "4": 33
}
```

**交互点完成状态**
```json
["1-1", "1-2", "1-3"]
```

## 用户体验流程

### 1. 选择场景
1. 进入体验馆首页
2. 浏览可用场景
3. 查看场景信息和进度
4. 点击场景卡片进入

### 2. 体验场景
1. 进入3D场景视图
2. 观察场景中的交互点标记
3. 点击交互点查看详情
4. 阅读内容后标记完成
5. 继续探索其他交互点

### 3. 完成体验
1. 完成所有交互点
2. 进度达到100%
3. 点击"生成体验证书"
4. 查看/下载/分享证书

## 快捷操作

### 场景列表页
- 点击场景卡片：进入场景
- 点击"我的进度"：查看所有场景进度
- 点击"返回首页"：返回平台首页

### 场景体验页
- 点击交互点：查看详情
- 点击"已了解"：标记完成
- 点击"退出场景"：返回列表
- 完成100%：生成证书

## 注意事项

1. **数据保存**: 所有进度保存在浏览器本地
2. **清除数据**: 清除浏览器数据会丢失进度
3. **3D场景**: 当前为占位符，实际3D需要后续开发
4. **性能**: 建议使用现代浏览器以获得最佳体验
