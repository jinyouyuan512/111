# 快速启动指南

## 🚀 启动项目

### 1. 安装依赖（首次运行）
```bash
cd frontend
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 访问应用
打开浏览器访问: **http://localhost:3000**

## 🎯 测试流程

### 完整购物流程测试
1. 访问 `/mall` 商城页面
2. 点击"加入购物车"添加商品
3. 点击右下角购物车图标进入 `/cart`
4. 选择商品，点击"去结算"
5. 自动跳转到 `/orders` 查看订单

### 体验馆测试
1. 访问 `/experience` 体验馆页面
2. 点击任意场景卡片进入
3. 点击场景中的交互点标记
4. 阅读内容后点击"已了解"
5. 完成所有交互点后生成证书

## 🔧 常见问题

### 页面空白
1. 确保开发服务器正在运行
2. 检查浏览器控制台是否有错误
3. 尝试清除浏览器缓存 (Ctrl+Shift+Delete)
4. 访问正确的端口 (3000)

### 样式丢失
1. 确保 Element Plus 正确安装
2. 重启开发服务器
3. 清除浏览器缓存

### 数据不显示
1. 打开浏览器控制台 (F12)
2. 执行: `localStorage.clear()`
3. 刷新页面 (F5)
