# 首页轮播图片目录

## 说明
此目录用于存放首页轮播的河北红色景点图片。

## 需要的图片文件

请将以下图片放入此目录：

1. **xibaipo.jpg** - 西柏坡革命圣地
   - 推荐内容：西柏坡纪念馆、七届二中全会会址等
   - 尺寸：1920x1080 或更高
   
2. **langyashan.jpg** - 狼牙山五壮士纪念地
   - 推荐内容：狼牙山主峰、五壮士纪念塔等
   - 尺寸：1920x1080 或更高
   
3. **ranzhuang.jpg** - 冉庄地道战遗址
   - 推荐内容：地道入口、地道战纪念馆等
   - 尺寸：1920x1080 或更高
   
4. **lidazhao.jpg** - 李大钊故居
   - 推荐内容：李大钊纪念馆、故居建筑等
   - 尺寸：1920x1080 或更高

## 图片要求

- 格式：JPG 或 PNG
- 尺寸：建议 1920x1080（16:9 比例）
- 大小：每张不超过 500KB
- 质量：清晰、有代表性

## 使用方法

图片放入后，在 `frontend/src/views/Home.vue` 中会自动引用：

```javascript
image: '/images/hero/xibaipo.jpg'
```
